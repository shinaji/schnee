# Contributing

## Release Versioning

Releases use CalVer tags in the form `vYYYY.M.D` or `vYYYY.M.D.N`.

- Use the UTC release date for `YYYY.M.D`.
- Add the optional `.N` suffix when more than one release is published on the
  same UTC day.
- Publishing to PyPI is triggered by creating a GitHub Release for one of those
  tags.
